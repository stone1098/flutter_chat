# 플러터 채팅 

유튜버 [코딩쉐프](https://www.youtube.com/@codingchef)님의 플러터 채팅 앱 강좌를 통해 만든 채팅 어플입니다.

firebase의 auth과 core를 사용하였고, flutter_chat_bubble를 사용하여 UI를 좀 더 꾸며주었습니다.
